# Cheerful Calculations

Modern bookkeeping website for restaurants and small businesses.

---

## Live Website

🌐 https://cheerfulcalculations.com

---

## Overview

Cheerful Calculations is a modern bookkeeping brand focused on helping restaurants and small businesses stay organized, reduce stress, and better understand their finances.

This repository serves as a public showcase of the project design, branding, and development work.

---

## Features

- Responsive modern UI
- Mobile-first design
- Light/Dark theme support
- SEO-friendly structure
- Custom branding system
- Resource and lead-generation pages
- MailerLite integration
- Clean component structure
- Performance-focused frontend architecture
